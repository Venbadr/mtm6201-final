### 3. Responsiveness  
Ensuring layouts looked good on all screen sizes was difficult and challenging
**Solution:** I used media queries and Bootstrap grid system to control layouts even tho that step was hard

### 4. Styling Conflicts with Bootstrap  
Some styles were overridden by Bootstrap defaults.  
**Solution:** I used custom CSS variables (`:root`) and more specific selectors to override Bootstrap properly.
